# v26 修复说明

本版修复：

1. 资产卡片下方三个小金额（资产 / 应收 / 负债）不再显示货币单位，避免设备显示放大时被挤压截断。
2. 所有金额为空时不再显示 `--`，改为 `0.000`，并统一保留 3 位小数。
3. 顶部净资产金额增加自适应缩放，字体放大或金额较长时不会直接溢出。
4. 密码锁触发时机降低敏感度：
   - 截图、下拉状态栏、普通系统弹窗返回，不再立即锁 App。
   - App 真正进入后台、Home 后返回、切到其它 App 后返回、重新打开、锁屏后返回，才会锁 App。

替换后执行：

```bash
flutter clean
flutter pub get
flutter analyze
flutter run
```
